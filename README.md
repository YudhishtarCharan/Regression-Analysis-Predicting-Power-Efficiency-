# Regression-Analysis-Predicting-Power-Efficiency-
Predicting device power efficiency by modeling complex interactions among numerical sensor metrics and categorical factors. Handling missing data, outliers, and feature engineering while applying Linear Regression, Decision Trees, Random Forest, Gradient Boosting, and XGBoost. Cleaning data, exploring patterns, encoding categories, training and validating models, and delivering reliable efficiency forecasts to support energy optimization and sustainability.
